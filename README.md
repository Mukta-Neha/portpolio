## Features
- **Responsive Design**: Optimized for desktop, tablet, and mobile devices.
- **Smooth Scrolling**: Anchor links navigate smoothly between sections.
- **Interactive Elements**: Hover effects on cards, progress bars for skills, and project links.
- **Sections Included**:
  - Home: Introduction with photo and bio.
  - About: Personal details, profile info, and interests.
  - Education: Academic background with cards for degrees.
  - Skills: Technical skills with progress bars and descriptions.
  - Projects: Showcase of personal projects (e.g., TODO List, Weather App, Calculator) with GitHub links.
  - Contact: Simple form for inquiries.
- **Dark Theme**: Modern black/orange color scheme for a sleek look.
- **Accessibility**: Semantic HTML and alt text for images.

## Technologies Used
- **Frontend**:
  - HTML5: Structure and content.
  - CSS3: Styling, animations, and responsiveness (with Poppins font from Google Fonts).
  - JavaScript: Basic interactivity (e.g., smooth scrolling, if extended).
- **Tools**:
  - GitHub: Version control and hosting.
  - GitHub Pages: Free static site hosting.
- **Libraries/Frameworks**: None (pure vanilla code for simplicity).
